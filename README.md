📌 Project Overview

AI Shopping Assistant is a modern full-stack e-commerce assistant designed to simplify online shopping through AI-driven conversations and personalized recommendations.

Instead of manually browsing products, users can interact with the assistant naturally using prompts like:

“Best smartwatch under ₹3000 with good battery life”

The system understands user intent, filters products intelligently, and provides recommendations with an interactive shopping experience.
